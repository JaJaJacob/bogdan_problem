Program bogdanek.exe ma za zadanie uruchomienie procesów zamykających Grand Theft Auto V w żądanym przez użytkownika momencie oraz 
zainicjowanie skryptu ponownego uruchamiania gry. Program robiony 
z myślą o ułatwieniu 
przeprowadzania napadu "Problem Bogdana"

---------WPROWADZ WŁASNE ZMIANY!---------

Aby program inicjujący skrypt uruchamiania gry zadziałał musisz wprowadzić swoją lokalizację programu uruchamiającego grę w Windowsie (PlayGTAV.exe). Jak to zrobić:

1. Edytuj PPM skrypt start_gta.cmd

2. po słowie start wpisz lokalizację gry np:
start C:\"ścieżka_do_gry"\"PlayGTAV.exe"

3. Po wpisaniu sciezki w nowym wierszu wpisz exit
