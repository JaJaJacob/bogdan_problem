Program bogdanek.exe was written to help with Bogdan Problem heist. Program is supposed to close gta, start gta. It is more comfortable than killing processes using Task Manager all.

---------MAKE YOUR OWN CHANGES!---------

To make programm running properly you must change the localization of PlayGTAV.exe file. How to do that:

1. Edit right mouse button script start_gta.cmd

2. After word 'start' type lozalication of a file, for example:
start C:\"path"\"PlayGTAV.exe"

3. After that in a new line type 'exit'
